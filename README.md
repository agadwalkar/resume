# Ashmita Gadwalkar — Resume Site

Personal resume website. Pure HTML/CSS/JS — no build step required.

## Viewing locally

```
open index.html
```

Or drag `index.html` into any browser.

## GitHub Pages

Push to a GitHub repo, go to **Settings → Pages → Deploy from branch (main / root)**, and the site will be live at `https://<your-username>.github.io/<repo-name>/`.

## Adding your PDF

Drop your resume PDF at:

```
assets/Ashmita_Gadwalkar_Resume.pdf
```

The "Download PDF" button in the nav and hero will automatically point to it.
